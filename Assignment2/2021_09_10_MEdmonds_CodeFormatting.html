Formatting Code

When you add code to an HTML file, normally the code will disappear and the tags you used will just format the code. For our website since we wanted to use the code as displayed text, we will need to use the <code> tags. These tags will ignore the formatting and just display the code fully.

Another issue with formatting code is that just using the tag puts all the code on one line. So, you will need to put your code tags within a <pre> tag to keep its shape.

Here’s how to use it:

<pre>

<code>

body {

background-color: blue;

}

h1 {

font-size: 35px;

}

</code>

</pre>

Here’s how the code will look on the webpage:




    body { 

        background-color: blue; 
    
    } 
    
    h1 { 
    
        font-size: 35px; 
    
    }

Designing your Code

If you want to change the colors of your code you can do that with CSS. In between the <code> tags you can add a <div> tag with a class attribute that with CSS and the div tag will disappear in your code segments.

For example,


    <style>

    .changeColor { 

        color: red; 

    } 

    </style>

    <div class="changeColor">body</div> { 

        background-color: blue; 

    }

Will look like:

 
body
 { 

    background-color: blue; 

}
